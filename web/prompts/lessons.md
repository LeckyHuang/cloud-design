## 2026-04-26 calc() 运算符空格
**问题**：`calc(100%-20px)` 运算符两侧无空格，浏览器静默忽略该属性，无报错极难排查。CSS 规范强制要求运算符周围有空格。
**修复**：改为 `calc(100% - 20px)`。
**预防**：写 calc() 时所有运算符（`+ - * /`）两侧必须留空格；无空格不报错但静默失效。

## 2026-04-26 SVG 撑满容器留白
**问题**：SVG viewBox 宽高比与容器不一致，默认 `preserveAspectRatio` 缩放后出现留白。用绝对定位 div 铺背景因脱离文档流，在弹性/网格布局中位置计算异常。
**修复**：SVG 填满容器用 `preserveAspectRatio="xMidYMid slice"`；背景改用 CSS `background-image` 或直接给 SVG 设 `width:100%; height:100%`。
**预防**：SVG 作背景前先对齐 viewBox 比例与容器；禁止用绝对定位 div 模拟背景，改用 CSS background 属性。

## 2026-04-26 绝对定位层叠陷阱
**问题**：`position:absolute` 元素覆盖了预期之外的元素，或位置偏移不符预期。根因是父容器未设 `position:relative`，绝对定位锚定到了更上层祖先。
**修复**：给直接父容器加 `position:relative`；若有 z-index 冲突再显式设置层级。
**预防**：用 `position:absolute` 前先确认父容器有 `position:relative`；出现覆盖异常先查祖先链，不要直接堆 z-index。

## 2026-04-26 HTML 重复 style 属性
**问题**：同一元素写了两个 `style=""` 属性，浏览器静默丢弃第二个，部分样式完全无效且无任何报错。
**修复**：合并为单个 `style` 属性，所有声明写在同一引号内。
**预防**：生成代码后用 Grep 搜 `style=.*style=` 快速扫描重复属性；同一元素禁止出现两个 style。

## 2026-04-26 object-position 方向口诀
**问题**：头像用 `object-fit:cover` 裁切后人物偏移，调 `object-position` 时方向搞反，越调越偏。
**修复**：口诀——**人物往哪偏，数值往同方向增大**（人物偏左 → left 值增大；人物偏上 → top 值增大）。
**预防**：先设 `50% 50%` 基准，再按口诀单轴微调；切勿凭感觉乱改两个轴。

## 2026-04-26 主题初始化四联检查
**问题**：修改默认主题时只改了 CSS 变量，遗漏其他初始化点，导致首屏主题与用户交互后状态不一致（主题"跳变"）。
**修复**：做四联检查——① CSS 变量默认值 ② JS 初始化赋值（读取 localStorage 或硬编码默认值）③ HTML 根元素 `data-theme` 初始属性 ④ Tweaks 面板默认选中项。
**预防**：改默认主题时必过四联检查，任一遗漏都会导致跳变；建议把四项写成注释锚点集中放在初始化函数旁。
