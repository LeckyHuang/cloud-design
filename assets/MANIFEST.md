## 使用说明

- **名称**：文件的语义名称（不含扩展名），供在设计对话中引用
- **类型**：image / icon / poster / reference / svg
- **描述**：简短说明用途（如「1920×1080 深色科技感背景图」）
- **路径**：相对于 `assets/` 目录的路径（如 `images/hero-dark.jpg`）
